# Sistema de Tabela de Pontos em Excel VBA

Este projeto foi desenvolvido para automatizar o preenchimento de pontos e ligações de equipamentos em projetos elétricos.

## 💡 Objetivo

Reduzir o tempo de montagem de tabelas de ligação através do reaproveitamento de dados já cadastrados.

## ⚙️ Funcionalidades

* Busca automática de pontos por equipamento
* Preenchimento automático de ligações
* Consulta de marca do componente
* Base de dados reutilizável

## 🛠️ Tecnologias

* Microsoft Excel
* VBA (Visual Basic for Applications)

## 📂 Estrutura do arquivo

* **Tabela de Ligações**: interface principal
* **Base de Dados**: armazenamento dos componentes
* **Lista de Componentes**: apoio de consulta
* **Como Usar**: instruções de utilização

## 📈 Resultado

O sistema reduz trabalho manual e aumenta a produtividade no preenchimento de tabelas técnicas.

## ⚠️ Observação

Os dados utilizados neste projeto foram adaptados para fins de portfólio.
